DuckDuckGo for Chrome - V2 Alpha

This is the V2 Alpha for the DuckDuckGo for Chrome extension. It adds:

  • Encryption - using HTTPS Everywhere
  • Tracker Blocking - using the Firefox/Disconnect tracker list
  • Social Blocking: blocks social requests and replaces Facebook and Twitter buttons - using the PrivacyBadger social widgets list and assets

This release is for internal testing.
